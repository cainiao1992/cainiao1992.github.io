Maven 笔记

2023-08-09

打包

# 正常打包
mvn package
# 跳过单元测试
mvn package -DskipTests
# 不编译单元测试类
mvn package -Dmaven.test.skip=true

设置阿里镜像源

编辑 $HOME/.m2/settings.xml 文件（没有的话，复制Maven安装目录/conf/settings.xml，到该目录下），找到 <mirrors> 插入以下代码。

		<mirror>
			<id>aliyunmaven</id>
			<mirrorOf>*</mirrorOf>
			<name>阿里云公共仓库</name>
			<url>https://maven.aliyun.com/repository/public</url>
		</mirror>

如在新版本中含有以下的 <mirror> ，需要删除。

    <mirror>
      <id>maven-default-http-blocker</id>
      <mirrorOf>external:http:*</mirrorOf>
      <name>Pseudo repository to mirror external repositories initially using HTTP.</name>
      <url>http://0.0.0.0/</url>
      <blocked>true</blocked>
    </mirror>

设置构建时的 Java 编译版本，Java 文件编码格式

    <properties>
        <maven.complier.source>11</maven.complier.source>
        <maven.complier.target>11</maven.complier.target>
        <project.build.sourceEncoding>UTF-8</project.build.sourceEncoding>
    </properties>
